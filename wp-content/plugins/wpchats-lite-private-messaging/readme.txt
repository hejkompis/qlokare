== Description ==

Instant chat & private messaging plugin with moderation, user profiles, preferences, emoticons and much more

<h3>Features</h3>
<strong>Messaging</strong>
<p>Send and receive private messages from users whom preferences allow you to contact.</p>

<strong>User profiles / Settings:</strong>
<p>User profiles, browse users, read their information (e.g online status) and keep in touch with them. Users can update anytime their settings and information: about, social profiles, notifications settings and user preferences..</p>

<strong>Conversation tools:</strong>
Users can benefit from handy conversation tools, allowing them to easily import media into messages and control those messages. Tools include:
<ul>
	<li>Message deleting: users can delete messages from their conversations.</li>
	<li>Message reporting: users can report messages to moderators anytime, and edit or delete those submitted reports if admin allows that.</li>
	<li>Add and search emoticons, more than 50 cool Facebook emoticons (emoji) which could be implemented in a single click.</li>
	<li>Conversation actions and much more..</li>
</ul>
<strong>50+ emoji:</strong>

<p>50+ cool Facebook emoticons which could be used within conversations..</p>

<strong>Moderators and Chat Moderation:</strong>
<p>Add moderators besides you, to moderate reported messages, delete them, contact users, moderate conversations, and much more..</p>

<strong>Admin dashboard:</strong>

<p>Update various settings, view stats, search messages, trash or restore messages, moderate queued messages and much more..</p>

== Installation ==

1. Upload the plugin to your plugins directory, or use the WordPress installer.
2. Activate the plugin through the \'Plugins\' menu in WordPress.
